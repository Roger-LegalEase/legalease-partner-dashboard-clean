# LegalEase Homepage — Complete Design Package

Customer-first commercial homepage for the LegalEase parent brand. Static site,
no build dependencies. Generated from `_build/gen.py`.

## What's in this package

```
legalease-homepage/
  index.html              production page (generated by gen.py)
  styles.css              all styling (Geist + Inter + Geist Mono, brand tokens)
  app.js                  three scroll/load animations, all reduced-motion safe
  assets/                 brand assets + real product assets (see below)
  fonts/                  9 self-hosted woff2 files (no external font CDN)
  _build/gen.py           the generator — edit + rerun to rebuild index.html
  DROP-IN-GUIDE.md        asset slot specs
  README.md               this file

legalease-homepage-standalone.html   single self-contained file (everything
                                      base64-inlined) for preview / artifact pane
```

## Brand

- Navy `#18186C`, deep blue `#0E0E40`, orange `#F04800`, skyline `#B8D8D8`,
  midnight `#07111F`, porcelain `#FAF7F0`.

## Known cosmetic notes (not bugs)

- Logo family: StartApart is a ribbon-S wordmark; RecordShield and Claim Coach
  are shields. In the 2x2 grid they read as a matched pair plus one outlier.
- Tagline inconsistency: StartApart's slot logo has no "by LegalEase" tagline;
  RecordShield and Claim Coach keep theirs. Their shield marks extend below the
  wordmark baseline, so the tagline can't be cropped without clipping the shield.
- `SHOT_STYLE = "shell"` wraps screenshots in browser chrome. Set to `"flat"` in
  `_build/gen.py` and rebuild for borderless cards.

## Rebuild

```
cd legalease-homepage
python3 _build/gen.py        # regenerates index.html from current assets/
```

The standalone is regenerated separately by inlining; see prior build notes.
